# zscalersdk-reactnative

React-Native bindings for Zscaler SDK

## Installation

```sh
npm install zscalersdk-reactnative
```

## Usage


```js
import { multiply } from 'zscalersdk-reactnative';

// ...

const result = await multiply(3, 7);
```
